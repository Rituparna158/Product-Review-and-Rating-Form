# **Product Review & Rating Form**
### Project Description:
This Project is a product review and rating form built using HTML , CSS  and JavaScript. Users can enter product details , star ratings , write reviews and submit the form with proper validation .

### Features:
1.Star Rating System<br>
2.Client side validation<br>
3.Real time error messages<br>
4.Responsive design<br>
5.Input restriction<br>
6.Terms and conditions validation

### Technologies used:
1.HTML<br>
2.CSS<br>
3.JavaScript es6 new feture<br>

### How to Run the project locally:
1.Open the project folder in VS code.<br>
2.Install live server extension<br>
3.Right click on the index.html<br>
4.Select "Open with Live Server".<br>
5.The Project will open in the browser.<br>
